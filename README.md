# 🎯 Life Audit App

> **"Most people are living someone else's life by accident. This app helps you live your own life on purpose."**

A comprehensive self-audit tool that helps you measure where your most finite resources—time, money, and attention—are actually going, versus where you want them to go.

## 📊 The Three Pillars

Life Audit is built on three interconnected pillars that provide a complete picture of how you're living:

### ⏱ **Time Pillar**
Track where your hours actually go—calendar, routines, sleep, daily rhythms.
- Manual time entry
- Google Calendar integration (auto-sync)
- Categorization into life areas
- Visual breakdowns and trends

### 💰 **Money Pillar**
See what your spending says about your values, not just categories.
- Manual expense entry
- Bank sync via Plaid API
- Spending mapped to life values
- INR currency support

### 🧠 **Attention Pillar**
Understand your screen time, media diet, and mental bandwidth.
- CSV upload (iOS/Android screen time)
- Manual attention logging
- Category-based analysis
- Reactive vs. intentional time tracking

## ✨ Key Features

- **🤖 AI-Powered Reports** - Weekly/monthly narrative reports generated by OpenAI that read like a thoughtful coach
- **📈 Live Dashboard** - Real-time charts and insights across all three pillars
- **🎯 Alignment Score** - Measures how well your actual behavior matches your stated priorities
- **🔄 Auto-Sync** - Google Calendar and bank transactions sync automatically
- **🔒 Privacy-First** - Your data stays yours. Supabase backend with Row Level Security
- **📱 Beautiful UI** - Modern, dark-themed interface built with Tailwind CSS

## 🚀 Quick Start

### Prerequisites
- Node.js 18+ and npm
- A Supabase account (free tier works)
- Optional: Google Calendar API credentials, Plaid API keys, OpenAI API key

### Installation

1. **Clone the repository**
```bash
git clone https://github.com/adityakale2002/Lifeaudit.git
cd Lifeaudit
```

2. **Install dependencies**
```bash
npm install
```

3. **Set up environment variables**

Create a `.env.local` file in the root directory:

```env
# Supabase (Required)
NEXT_PUBLIC_SUPABASE_URL=your-supabase-url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your-supabase-anon-key

# OpenAI (Required for AI reports)
OPENAI_API_KEY=your-openai-api-key

# Google Calendar (Optional)
GOOGLE_CLIENT_ID=your-google-client-id
GOOGLE_CLIENT_SECRET=your-google-client-secret
GOOGLE_REDIRECT_URI=http://localhost:3000/api/calendar/callback

# Plaid Bank Sync (Optional)
PLAID_CLIENT_ID=your-plaid-client-id
PLAID_SECRET=your-plaid-secret
PLAID_ENV=sandbox

# App URL
NEXT_PUBLIC_APP_URL=http://localhost:3000
```

4. **Set up the database**

Go to your Supabase project → SQL Editor and run the schema from `supabase/schema.sql`

5. **Run the development server**
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## 📖 Usage Guide

### Getting Started

1. **Sign up** and create your account
2. **Set your priorities** in Settings → Priorities (what matters most to you?)
3. **Start logging data**:
   - Manual entry for quick logs
   - Connect Google Calendar for automatic time tracking
   - Connect your bank via Plaid for automatic expense tracking
   - Upload screen time CSV for attention tracking

### Manual Entry

Navigate to the **Entry** page to manually log:
- **Time blocks** - "Family dinner, 2 hours"
- **Expenses** - "Groceries, ₹3000"
- **Attention** - Coming soon

### Integrations

#### Google Calendar Sync
1. Go to Settings → Connections
2. Click "Connect" on Google Calendar
3. Authorize the app
4. Your events will auto-sync and be categorized

#### Bank Sync (Plaid)
1. Go to Settings → Connections
2. Click "Connect" on Bank & Finance
3. Follow Plaid Link flow (sandbox mode for testing)
4. Transactions will auto-sync weekly

#### Screen Time Upload
1. Export screen time from iOS/Android as CSV
2. Go to Attention page or Settings
3. Upload your CSV file
4. Data will be visualized immediately

**CSV Format:**
```csv
Date,App Name,Category,Hours
2026-04-01,Instagram,Social,2.5
2026-04-01,Chrome,Productivity,1.2
```

### AI Reports

1. Log data for at least a few days
2. Navigate to **Report** page
3. Click "Generate Report"
4. Get a personalized narrative with insights and recommendations

## 🎨 Features in Detail

### Life Buckets

All data is categorized into 7 life areas:
- 👨‍👩‍👧 **Family** - Relationships and loved ones
- 💼 **Work** - Career and professional life
- 💪 **Health** - Physical and mental wellbeing
- 📚 **Growth** - Learning and personal development
- 🎮 **Leisure** - Entertainment and fun
- 📋 **Admin** - Errands and life maintenance
- 😴 **Sleep** - Rest and recovery

### Alignment Score

The app calculates how well your actual behavior matches your stated priorities:
- Compares time spent vs. priority ranking
- Highlights gaps between intention and reality
- Updates in real-time as you log data

### Privacy & Security

- 🔒 Row Level Security (RLS) - You can only see your own data
- 🔐 Encrypted connections (HTTPS)
- 🚫 No tracking or analytics
- 💾 Your data stays in your Supabase instance
- 🔑 API keys stored securely in environment variables
